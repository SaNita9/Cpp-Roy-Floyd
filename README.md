# Cpp-Roy-Floyd
## Overview
  Acest proiect reprezinta o implementare a algoritmului Roy-Floyd, folosit pentru a determina distanta minima dintre doua noduri intr-un graf ponderat orientat (arcele au fiecare cate un cost de parcurgere).
  Algoritmul citeste datele de intrare dintr-un fisier de input, calculeaza distanta minima dintre oricare doua noduri si scrie rezultatele intr-un fisier de output sub forma unei matrice de distante.
## Structura roy-floyd.in:
* n – numarul de noduri
* m – numarul de arce
* m arce sub forma tripletului (x, y, z) 
  * x – nodul de inceput
  * y – nodul destinatie
  * z – costul
## Mentiuni roy-floyd.out:
Distanta de la un nod la el insusi se considera 0; 
Daca un exista drum de la un nod la altul, valoarea din matrice corespunzatoare drumului se considera -1.
